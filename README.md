# OS Simulation Project - Spring 2026

A Python-based simulation of core Operating System functions, including CPU Scheduling, Memory Allocation, and Page Replacement, featuring full visualization using Matplotlib.

## 🚀 Features
- **CPU Scheduling:** FCFS algorithm with Gantt Chart visualization.
- **Memory Allocation:** First-Fit algorithm with Internal Fragmentation analysis.
- **Page Replacement:** FIFO algorithm with step-by-step Frame Grid visualization.
- **Dynamic Input:** No hardcoded values; all data is provided by the user at runtime.

## 🛠️ Built With
- Python 3.x
- Matplotlib (for Visualization)
- Jupyter Notebook
